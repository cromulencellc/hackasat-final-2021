Readme file for the gen_app_config_files script.

This script will generate the base config files for the new applications listed in the command. It will work with a fresh app created by the gen_app_code cfs tool.

1. Got to your path/to/cosmos/config/targets directory
2. Execute the Python script located in path/to/cosmos/scripts/gen_app_config_files/gen_app_config_files.py
3. Follow the usage description to generate the base code for your new applications.  

   Example: > $ python /path/to/cosmos/scripts/gen_app_config_files/gen_app_config_files.py OUTDIR . APPS newApp1

Note: Pay close attention to how you name your applications at this step since the specified application names will be used as part of the file 
names
